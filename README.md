Eco-Friendly Online Store

A simple and responsive web project showcasing eco-friendly products. Users can browse items, add them to a cart, and complete purchases with a confirmation message.

Features

* Home, Shop, About, and Contact pages
* Product listing with images
* Add-to-Cart functionality
* Dynamic cart with total price calculation
* Buy button → "Successfully bought!" popup
* Clean UI with responsive layout

Technologies Used

* HTML
* CSS
* JavaScript
